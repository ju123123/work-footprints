# 工作足迹
> 记录每一份投入，沉淀每一次成长。工作足迹是一个用于记录、整理、沉淀和复盘工作内容的系统，帮助个人或团队更清晰地回顾每日工作、跟踪项目进展、汇总阶段成果，并形成可检索、可复用的工作知识库。

## 核心目标 
- 记录每日工作内容、进展与问题
- 按项目、模块、任务维度沉淀工作过程
- 支持日报、周报、月报快速汇总
- 记录成果、风险、阻塞项与复盘结论
- 让零散的工作记录逐步沉淀为可追溯的工作足迹

### 1. 工作记录 - 新增每日工作记录 
- 记录工作事项、完成情况、耗时与备注
- 支持草稿、发布与编辑
- 支持按日期查看工作内容 

### 2. 项目与任务关联 
- 工作记录关联项目、模块和任务
- 查看项目的工作进展与历史记录
- 汇总项目阶段成果、待办事项和风险项 

### 3. 周报与总结 
- 按时间范围自动汇总工作记录
- 生成日报、周报、月报内容
- 支持手动补充重点成果、问题与下阶段计划 

### 4. 工作复盘 
- 记录问题、阻塞原因和解决过程
- 沉淀经验、改进措施和可复用方案
- 支持按标签检索历史复盘内容 

### 5. 检索与统计 
- 按日期、项目、标签、关键词检索
- 统计工作投入、项目进展和成果分布
- 支持查看个人或团队工作轨迹

## 主要页面规划
- 首页：今日工作、近期项目、待办事项、工作统计
- 工作记录：按日期查看和编辑每日工作内容
- 项目管理：项目列表、模块、进展与关联记录
- 工作总结：日报、周报、月报生成与管理
- 复盘沉淀：问题记录、经验总结、知识检索
- 数据看板：工作量、项目投入、成果与趋势统计 

## 技术栈 
- 后端：Spring Boot、MyBatis-Plus、MySQL、Redis、Spring-ai-alibaba
- 前端：Vue 3、TypeScript
- 部署：Docker Compose、Nginx
- AI能力：AI 总结、AI 周报生成、AI 风险识别、AI 工作复盘建议

## 后续规划

- 根据每日记录自动生成周报、月报
